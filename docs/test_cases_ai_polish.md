## 一、核心逻辑（4 种组合）

### TC-001: AI 开 + 词汇开 — LLM 结合词典术语纠正
- **前置条件**: AI 润色开关打开，专业词汇开关打开，词典中有词条 `语义分割 (wrong: 鱼米分割)`，LLM 配置正确
- **操作**: 输入 ASR 文本 `这个模型的鱼米分割效果很好`
- **期望结果**: LLM 收到的请求中包含 `<vocab_hints>语义分割</vocab_hints>`，最终输出文本将"鱼米分割"纠正为"语义分割"
- **分类**: 核心逻辑

### TC-002: AI 开 + 词汇关 — 纯 LLM 润色无术语提示
- **前置条件**: AI 润色开关打开，专业词汇开关关闭，LLM 配置正确
- **操作**: 输入 ASR 文本 `嗯那个那个我觉得这个方案可以`
- **期望结果**: LLM 收到的请求中不包含 `<vocab_hints>` 标签，LLM 对口语冗余和标点进行润色
- **分类**: 核心逻辑

### TC-003: AI 关 + 词汇开 — 精确字符串替换回退
- **前置条件**: AI 润色开关关闭，专业词汇开关打开，词典中有词条 `wrong: 机器学系 → correct: 机器学习`
- **操作**: 输入 ASR 文本 `机器学系是人工智能的一个分支`
- **期望结果**: 不调用 LLM，直接将"机器学系"替换为"机器学习"，输出 `机器学习是人工智能的一个分支`
- **分类**: 核心逻辑

### TC-004: AI 关 + 词汇关 — 原文直出
- **前置条件**: AI 润色开关关闭，专业词汇开关关闭
- **操作**: 输入 ASR 文本 `嗯那个我觉得这个方案可以的`
- **期望结果**: 不调用 LLM，不做任何替换，输出与输入完全一致
- **分类**: 核心逻辑

### TC-005: AI 开 + 词汇开 — 词典为空时等效于纯 LLM 润色
- **前置条件**: AI 润色开关打开，专业词汇开关打开，但所有行业词典未勾选且无自定义词条
- **操作**: 输入 ASR 文本 `今天天气不错`
- **期望结果**: vocab hints 为空列表，LLM 请求中 `<vocab_hints>` 为空或不包含该标签，LLM 正常润色
- **分类**: 核心逻辑

### TC-006: AI 关 + 词汇开 — 词典为空时不改原文
- **前置条件**: AI 润色开关关闭，专业词汇开关打开，但词典无任何词条
- **操作**: 输入 ASR 文本 `今天天气不错`
- **期望结果**: 无替换规则可匹配，原文直出
- **分类**: 核心逻辑
